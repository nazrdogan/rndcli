# Commands

### See list of available devices

``` 

rndcli

```
you will see list of device.Select one of them


### Run  with  selected device

``` 
rndcli -d option

```
##### Example

``` 
rndcli -d 2

```
option : 0 - available devices count

# Installation

``` 
 npm install rndcli -g
 
```
